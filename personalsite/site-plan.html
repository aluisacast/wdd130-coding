Projects

White Water Rafting Site / HTML & CSS: Webpage that offers services for a white-water rafting company. Site Plan, where the Logo, color palette and font where chosen. Design of the Wireframes and Site Plans for the child pages.

Wordle type game / Python: Game where a user can guess a word from a list. It compares the amount of words the user places in the input and the amount of letters in the guess word. It gives hints, of the placement of the letters. Perceives the number of guesses the user needed to guess the word. It will continue to loop until the user wants to stop playing.

Choose your own adventure game / Python: It is a story where the user can choose what the adventure’s hero does in their quest to find a treasure. Includes ASCII art in some of the decisions the user chooses. While detecting if the input placed by the user when choosing an option is valid.

Data analysis, Life expectancy / Python: Reading and analysis of the chosen variables of the user for the 1523 -2019 World Life Expectancy. The user can compare the Life Expectancy by country, year, or the overall max and min life expectancy in the world.
